Purpose

This is a project is made to spread awareness on the different polling places in PG County

How to use

To use this dynamic web application the user needs to type in the polling address of interest into the search bar and then click on the 'Catalog' button to generate a list of the different polling places around that location. Then if the user clicks the 'Plot' button, the map itself would be populated to with location icons that indicate the different polling locations near the address the user entered.
